# i3o (Insta360 Organize)

Insta360で撮影した動画・プロキシファイル群を、ファイル名から撮影日を抽出して自動でディレクトリ（`YYYY-MM-DD`）ごとに仕分けするCLIツールです。
Go言語で記述されており、依存関係のないシングルバイナリとして高速に動作します。

## 🚀 Features

- **自動日付抽出**: `VID_YYYYMMDD_...` のようなファイル名から撮影日を正確に抽出します。
- **マルチフォーマット対応**: 標準の動画ファイル（`.mp4`）だけでなく、Insta360独自の高画質ファイル（`.insv`）やプレビュー用ファイル（`.lrv`）、Proモードの動画にも対応しています。
- **安全な処理**: 既に同名のディレクトリが存在する場合は再利用し、同名の「ファイル」が存在して競合する場合は安全にスキップします。

## 📦 Installation

Go環境（1.17以上）がインストールされている場合は、標準の `go install` コマンドでシステムにインストールできます。

```bash
# リポジトリをクローンしてインストールする場合
git clone <your-repository-url>/i3o
cd i3o
go install .
```
